                         INTERACTION FLOW

1. Admin opens Area Assignment page
   └─► Statistics cards display current counts

2. Admin searches for user "John"
   └─► User list filters to show matching users

3. Admin clicks on "John Doe" card
   └─► User card highlights with blue border
   └─► Area panel loads John's current assignments
   └─► Statistics update to show assigned count

4. Admin searches areas "ABC"
   └─► Area cards filter to show matching areas

5. Admin clicks area cards to toggle selection
   └─► Checkbox animates checked/unchecked
   └─► Selected count updates
   └─► Progress bar animates

6. Admin clicks "Select All"
   └─► All visible areas get selected
   └─► Progress bar fills

7. Admin clicks "Save Assignments"
   └─► Button shows loading spinner
   └─► API call sends data to backend
   └─► Success toast notification appears
   └─► Assignments are saved
